Formatting Tags

Element Name Element Description
Bold b This is a physical tag, which is used to bold the text written between
Strong strong This is a logical tag, which tells the browser that the text is important.
Italic i This is a physical tag which is used to make text italic.
Emphasize em This is a logical tag which is used to display content in italic.
Mark mark This tag is used to highlight text.
Underline u This tag is used to underline text written between it.
Strikethrough strike This is a physical tag which is used to make text italic.
Superscript sup It displays the content slightly above the normal line.text
Subscript sub It displays the content slightly below the normal line.text
